## cvebase

Search engine for security professionals. 1.7M+ documents across 110 sources in one semantic search.

**[cvebase.io](https://cvebase.io)**

### What it does

- **One search** across NVD, ExploitDB, Sigma, Nuclei, YARA, MITRE ATT&CK, vendor advisories, and 100+ other sources
- **CVE pages** with CVSS, EPSS score, CISA KEV status, affected versions, exploits, detection rules
- **Semantic search** — search by concept, not just keywords
- **Free API** — 100 requests/day, no signup required

### Sources

Vulnerability databases, exploit repositories, detection rules, vendor advisories, threat intelligence, research papers, and community discussions — aggregated and cross-linked.
